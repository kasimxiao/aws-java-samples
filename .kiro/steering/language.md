# 语言规范

本项目所有文本内容统一使用中文。

## 代码注释

- 类注释、方法注释、行内注释均使用中文
- Javadoc 文档使用中文描述
- TODO/FIXME 等标记说明使用中文

## 交互沟通

- 与用户的所有对话使用中文
- 错误提示和建议使用中文
- 代码审查反馈使用中文

## Git 提交

- commit message 必须使用中文
- 提交信息格式：`<类型>: <简短描述>`
- 类型包括：功能、修复、重构、文档、测试、配置等

### 示例

```
功能: 添加 EC2 实例创建功能
修复: 修复 IAM 角色绑定失败问题
重构: 优化 SSM 命令执行逻辑
文档: 更新 README 使用说明
测试: 添加 SageMaker 部署单元测试
```
