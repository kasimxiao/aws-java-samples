# 产品概述

AWS Java 示例项目 - 基于 AWS SDK for Java 2.x 的云资源管理工具集。

## 核心功能

- EC2 实例管理：创建、启动、停止、终止、重启实例
- AMI 镜像管理：创建、复制、共享、删除镜像
- IAM 角色管理：实例配置文件绑定与解绑
- SSM 命令执行：远程命令执行、S3 挂载
- DCV 远程桌面：生成免密登录 URL
- SageMaker 机器学习：模型训练、部署、监控

## 设计目标

- 提供统一的 AWS 资源管理入口（Ec2Manager）
- 封装 AWS SDK 复杂性，简化调用
- 支持配置文件驱动，便于环境切换
